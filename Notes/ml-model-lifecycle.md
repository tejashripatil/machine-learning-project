#what is Machne learning
-Machine  learning is sub set of AI that uses Algorihtms and feature engineering.
-teaches computers to learn from data, identify patterns and  make decisions
-does not rely on fixed algorithm, uses computational methods for learning

#paradigms
1.Supervised learning models- train on labled data
2.unsupervised learning models- train on unlabled data
3.semi-supervised learning models- relatively small data set labeled
4.Reinforcement learning models- learns from feedback

# techniques:
1. Classification- predicts class/category(short or tall) (supervised )
2. Regression/estimation- predicts contihous values(price of apartment)(supervised)
3. Clustering- similar data (unsupervised)
4. Association- events/items that co-occur
5. Anamoly detection- e.g credit card frauds
6. sequence mining: predicts next event
7. Dimention Reduction
8. Recommendation


#Processes involved

5. deployment

-Lifecycle is iterative
-data collection and data preparation is called ETL Process: extract, transform and load.
